### install from npm

    $ npm install  opcua-commander -g
    $ opcua-commander -e opc.tcp://localhost:26543 
        

### install from source


    $ git clone https://github.com/node-opcua/opcua-commander.git
    $ cd opcua-commander
    $ npm install
    $ node index.js  -e opc.tcp://localhost:26543
